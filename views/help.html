Excluding an individual blog post

If you write a blog post that is 'off topic' and don't want it to be included in the Aus GLAM Blogs database or tweeted/tooted, you can exclude it from being ingested by assigning one of the following tags to the post using your blogging software:

  • notglam
  • notglamblogs
  • notausglamblogs
